RE Wysiwyg Profile Feature
----------------------------------------------------------------------
This feature create three Text Formats:
	- Full HTML
	- Filtered HML
	- Plain text

And assigns a configure TinyMCE Wysiwyg Profile to Full HTML.

Please also ensure that you have downloaded the most recent version of TinyMCE and placed it in sites/all/libraries.
Please also patch the Wysiwyg 7.x-2.1 with http://drupal.org/files/0001-feature.inc-from-624018-211.patch, which can be found on this thread - http://drupal.org/node/624018#comment-5098162
